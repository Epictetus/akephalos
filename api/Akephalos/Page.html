Overview

Akephalos::Page wraps HtmlUnit’s HtmlPage class, exposing an API for interacting with a page in the browser.

def find(selector)
  nodes = @_page.getByXPath(selector).map { |node| Node.new(node) }
  @nodes << nodes
  nodes
end
